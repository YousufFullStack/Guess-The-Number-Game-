# 🎯 Guess the Number Game

A fun and interactive **Guess the Number Game** built using **HTML**, **CSS**, and **JavaScript**. The game randomly selects a number between **1 and 100**, and the player has **10 attempts** to guess it correctly.

## 🚀 Features

* 🎲 Random number generation between **1 and 100**
* 🔢 User-friendly number input
* 📋 Displays previous guesses
* ⏳ Tracks remaining attempts
* 📈 Provides hints:

  * Too High
  * Too Low
  * Correct Guess
* 🔄 Start a new game without refreshing the page
* 📱 Responsive and clean user interface

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
